rSkybox Sample Application

(View source to see how rSkybox apps are put together.)

Steps to integrate rSkybox into your HTML5 application:

  1. Sign up for rSkybox.
  2. Create an application.
  3. Download the automatically generated JavaScript files, which will include your application's settings:
    • Library: rskybox.js
    • Config: rskybox-config.js
  4. Include your JavaScript files in your HTML5 file(s).
  5. Add log statements to your JavaScript code to capture incidents on the rSkybox server.
  6. Customize your rSkybox configuration file to add information appropriate to your application.
